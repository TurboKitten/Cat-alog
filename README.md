# Cat-alog
Proyecto "Cat-alog" de Diego Garrido y Matias Duncker 

El proyecto cuenta con el sistema Model, Controller, Repository, Service 

Se ha añadido Spring Security (JWT) al proyecto, al igual que "Maven" y "Dockers

El proyecto tambien cuenta con "CatAPI" externa, la cual permite obtener una foto aleatoria de un gato 

Se ha añadido tambien Swagger UI al proyecto: http://localhost:8080/swagger-ui/index.html#/usuario-_-gatos-controler/listarUsuarios
